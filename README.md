`shinyRAT` is a R Shiny app that uses `spaceRAT` to project samples into
a scaffold space.

## Installation

You can install the development version of shinyRAT from GitHub with:

``` r
# install.packages("remotes")
remotes::install_github("shdam/shinyRAT")
```

## Example

### Load Shiny app

``` r
library(shinyRAT)

run_app()
```
